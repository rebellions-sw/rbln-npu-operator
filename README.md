# Rebellions NPU Operator Helm Repository

This repository hosts the Helm chart for deploying the **Rebellions NPU Operator**, which automates device discovery, management, and monitoring of Rebellions NPU devices on Kubernetes clusters.

The artifacts published in this branch are intended to be consumed as a **Helm Chart Repository**, backed by GitHub Pages.
